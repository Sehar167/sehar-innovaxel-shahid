# Innovaxel Project

## Setup Instructions

1. Clone the repo  
   `git clone https://github.com/Sehar167/sehar-innovaxel-shahid.git`

2. Switch to `dev` branch  
   `git checkout dev`

3. Install dependencies  
   `pip install -r requirements.txt`

4. Run the app  
   `python app.py`

## Notes

- All code is in the `dev` branch  
- `main` contains only this README
